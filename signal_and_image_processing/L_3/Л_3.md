
## Лабораторна робота №3

**Мета:** Метою даної лабораторної роботи є дослідження методів
відновлення перекручених зображень як за відсутності шумів, так і з ними.

<center>Хід виконання</center>

У ході виконання цієї лабораторної роботи відбулось знайомство з методами відновлення зображень.

Завдання на самостійну роботу 

1-2.Завантажити з бібліотеки MATLAB тестове зображення та відобразити його.
<img width="747" height="624" alt="image" src="https://github.com/user-attachments/assets/7717aae0-7536-457b-b802-f27faa34c253" />


3.Здійснити процедуру перекручення зображення змінюючи параметри LEN і THETA.
```
LEN = 25;      % довжина розмиття

THETA = 45;    % кут

PSF = fspecial('motion', LEN, THETA);

I_blur = imfilter(I, PSF, 'conv', 'circular');

```
4.Відобразити перекручене зображення.
<img width="734" height="622" alt="image" src="https://github.com/user-attachments/assets/fa40a2cd-acfd-46c9-8ffd-7b9cccae7544" />

5.Виконати процедуру відновлення зображення.
```
I_restore = deconvwnr(I_blur, PSF);

```

6.Відобразити зображення після відновлення.
<img width="733" height="659" alt="image" src="https://github.com/user-attachments/assets/28559440-5169-4ecc-ae5d-70db0559d1e9" />

7.Виконати зашумлення початкового зображення та повторити пункти 2-6.
```
I_noise = imnoise(I, 'gaussian', 0, 0.002);

I_blur2 = imfilter(I_noise, PSF, 'conv', 'circular');

I_restore2 = deconvwnr(I_blur2, PSF);
```
<img width="1490" height="463" alt="image" src="https://github.com/user-attachments/assets/b6af20ab-597b-43c4-823b-e0cdaa69e58e" />

Вийшло не якісне зображеення, тому змінила параметри зашумлення для кращого зображення : 
```
NSR = 0.01;   % коефіцієнт шуму

I_restore = deconvwnr(I_blur, PSF, NSR);

```

8.Відповісти на наступні запитання:
- Поясніть процес формування зображення.

  Процес формування зображення полягає у перетворенні світлового сигналу, що надходить від об'єкта, у цифровий сигнал. У процесі отримання зображення можуть виникати спотворення, шум та розмиття.
- Опишіть модель формування зображень та лінійних перекручень.

  Модель формування зображення можна записати як - g = h * f + n, де f — початкове зображення, h — функція перекручення, n — шум, g — отримане зображення
Лінійні перекручення виникають через розмиття, рух камери або дефокус.
- Поясніть процес відновлення перекрученого зображення.

  Відновлення виконується за допомогою зворотної фільтрації або вінерівського фільтра.
  Метод Wiener враховує шум і дозволяє отримати більш точне відновлення зображення.
