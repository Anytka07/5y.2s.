методи стиснення зображень та ознайомитися з основними з них. 

<center>Хід виконання</center>
<br>У ході виконання роботи було досліджено метод стиснення зображень на основі DCT

<br>Завдання на самостійну роботу


<br> 3. З використанням функції dct2 виконайте дискретне косинусне перетворення зображень. Відобразіть результат ДКП у вигляді зображення з
використанням функції imshow. Під час відображенні масиву ДКП-коефіцієнтів використовуйте логарифмічний масштаб.
DCT - це дискретне косинусне перетворення що переводить зображення з просторової області (пікселі) у частотну область (коефіцієнти)
```
J1 = dct2(I1g);
J2 = dct2(I2);
J3 = dct2(I3g);
```

<img width="1163" height="806" alt="image" src="https://github.com/user-attachments/assets/e5ac3370-dce4-44bc-9e37-1f29b54e640a" />
Отриманий ДКП спектр: найбільші (найбільш яскраві) його значення зосереджені в лівому верхньому куті (це
низькочастотні складові ДКП), уся ж інша частина масиву (права нижня його частина - високочастотні складові) заповнена відносно невеликими
значеннями (точками відносно невисокої яскравості).

<br>5-6. З використанням процедури цілочислового ділення/множення J = N*round(J*N); виконайте квантування результатів ДКП для різних значень
кроку квантування N. Поясніть, як працює ця процедура, і що отримаємо в результаті. Відобразіть результати у вигляді зображення 
```
N1 = 10;
N2 = 30;

J1_q1 = N1 * round(J1 / N1);
J1_q2 = N2 * round(J1 / N2);
```
<img width="1164" height="803" alt="image" src="https://github.com/user-attachments/assets/51ccde11-5109-4877-bedf-65b359d8d5ab" />
Квантування призводить до занулення малих коефіцієнтів, що зменшує обсяг інформації. Тому бачимо більше чорного

<br>7. З використанням функції idct2 відновіть зображення за квантованим ДКП-спектром для різних значень кроку квантування. Поясніть отримані
результати. 
```
K_q1 = idct2(J1_q1);
K_q2 = idct2(J1_q2);
```
N маленьке - якість норм, N велике - зображення псується.

Зі збільшенням кроку квантування зростає ступінь стиснення, але погіршується якість зображення.

<br>8. Поясніть, яка мета досягається квантуванням коефіцієнтів ДКП.

Квантування використовується для зменшення обсягу даних шляхом видалення малозначущих компонентів зображення.
головна ідея:
менше даних, швидша передача зображення, менше пам’яті

<br>9. Чи можливо добитися аналогічної мети й результату, квантуючи вихідне зображення, а не коефіцієнти його ДКП? Перевірте це на практиці,
квантуючи з використанням процедури I = (round(J/n))*n вихідні зображення й відобразіть отриманий результат.
```
I_quant = round(double(I1g)/20)*20;

figure('Name','Квантування зображення')
subplot(1,2,1); imshow(I1g); title('Оригінал');
subplot(1,2,2); imshow(I_quant,[]); title('Квантування пікселів');
```
<img width="1164" height="816" alt="image" src="https://github.com/user-attachments/assets/d9880c2a-b29f-41c3-a284-0199d5091271" />
Квантування пікселів менш ефективне, ніж квантування DCT, оскільки не враховує частотну структуру зображення.
Втрачається якість сильно, з’являються "сходинки" та немає розділення на важливе/неважливе

<br>10. Які недоліки ви бачите в стисненні зображень із використанням його ДКП і квантування коефіцієнтів ДКП? 

Основним недоліком є втрата якості зображення при високому рівні стиснення та поява артефактів.

**Висновки**

У роботі було досліджено метод стиснення зображень на основі DCT. Встановлено, що основна енергія зображення зосереджена у низьких частотах, 
що дозволяє ефективно виконувати стиснення шляхом квантування коефіцієнтів.
